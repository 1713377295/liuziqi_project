## <p align="center">可行性分析</p>
  由于六子棋发明不久，目前在学术界引起的关注较少。但是六子棋是一种新颖、有趣、易学难精的棋。由于每人每次下两步(第一步除外)，所以它的对攻性强，可变性大，棋路开阔，且又免去了五子棋繁杂的禁手，因此极受大众欢迎。在公平性上进行拓展，就会发现，如果想要实现人机对弈的绝对公平，就需要六子棋算法可以精准的算到每一步的落点并且和人的思想同步。
